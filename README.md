[![Portfolio](https://img.shields.io/badge/Portfolio-Live-brightgreen)](https://yasserhegazy.github.io/Portfolio)

[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-38B2AC)](https://tailwindcss.com/)

Modern, responsive portfolio website showcasing my projects, skills, and experience as a software developer.

## ✨ Features

- 🌍 Bilingual support (English & Arabic with RTL)
- 🌓 Dark/Light mode toggle
- 📱 Fully responsive design
- 🎨 Smooth animations with Framer Motion
- ⚡ Optimized performance
- ♿ Accessible (WCAG compliant)

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

## 📂 Project Structure

```
├── app/                  # Next.js App Router pages
├── components/          # React components
├── contexts/            # React contexts (Theme, Language)
├── translations/        # Bilingual content (en, ar)
├── lib/                 # Utilities
└── public/              # Static assets
```
